# IfGPT Dataset

## Objectives of the project IfGPT

The **IfGPT Dataset** is developed within the project **IfGPT: Infrastructure for Fine-tuning Pre-trained Large Language Models** which aims to establish a freely accessible infrastructure for the selection and pre-processing of large datasets for Bulgarian as well as tailored data for particular industries and fine-tuning suitable freely available large language models for specific purposes.

## IfGPT Dataset

IfGPT is a large dataset of Bulgarian which encompasses texts from various sources. The texts are supplied with metadata description allowing for filtering and selection of suitable texts for domain specific and tasks specific training and fine-tuning. 

Here we include only cleaned and deduplicated data from the IfGPT Dataset with licenses allowing redistribution.

This collection includes three Bulgarian-language datasets covering legal, multi-domain, and encyclopedic text.

### MARCELL (BG)

| Field | Value |
|---|---|
| **Language** | Bulgarian |
| **Domain** | Legal |
| **Size** | ~25,000 texts / 3.28M sentences / 45M tokens |
| **Source** | Bulgarian State Gazette |
| **Licence** | Public Domain |
| **Download** | [Link](https://huggingface.co/datasets/DCL-IBL/IfGPT-OPEN-Dataset/blob/main/IfGPT_MARCELL.tar.gz) |
